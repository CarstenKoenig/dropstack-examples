# DROPSTACK Examples

## Deploy via [https://dropstack.run](https://dropstack.run)

```bash
npm i -g dropstack-cli
```

## Sign in

```bash
dropstack login https://my-cloud.domain.com
```

## Deploy

```bash
dropstack deploy
```
